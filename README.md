# 🏥 Patient Classifier

A Machine Learning-powered healthcare application that predicts whether a patient should be **Sent Home** or **Kept Under Care** based on key health parameters. The system uses a Decision Tree Classifier trained on patient vital data and provides predictions through a simple and user-friendly Flask web interface.

---

# 🚀 Features

* Interactive and responsive web interface
* Real-time patient classification
* Machine Learning-based decision support
* Flask backend integration
* Fast prediction results
* Simple and lightweight architecture
* Easy deployment and customization

---

# 📊 Input Parameters

The model makes predictions using the following patient metrics:

| Parameter      | Description                      |
| -------------- | -------------------------------- |
| Temperature    | Patient body temperature         |
| Oxygen Level   | Blood oxygen saturation (%)      |
| Blood Pressure | Patient blood pressure           |
| Comfort Score  | Patient comfort assessment score |

---

# 🧠 Machine Learning Model

### Algorithm Used

* Decision Tree Classifier (Scikit-Learn)

### Output Classes

| Prediction | Meaning            |
| ---------- | ------------------ |
| 1          | ✅ Sent Home        |
| 0          | ⚠️ Keep Under Care |

### Model Workflow

1. User enters patient health metrics.
2. Frontend sends the data to the Flask backend.
3. The trained Decision Tree model processes the input.
4. Prediction is generated.
5. Result is displayed instantly on the webpage.

# ⚙️ Installation & Setup

## 1. Clone the Repository

```bash
git clone https://github.com/Mehul-ui141/Patient-Classifier.git
cd Patient-Classifier
```

## 2. Create a Virtual Environment

```bash
python -m venv venv
```

## 3. Activate the Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux/macOS

```bash
source venv/bin/activate
```

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

## 5. Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
