# 🏥 Patient Classifier

A Machine Learning-powered healthcare application that predicts whether a patient should be **sent home** or **kept under medical care** based on key health parameters. The system uses a Decision Tree Classifier trained on patient vital data and provides predictions through a simple Flask web interface.

## 🚀 Features

* Interactive and responsive web interface
* Real-time patient classification
* Machine Learning-based decision support
* Flask REST API integration
* Fast prediction results
* Simple deployment and setup

---

## 📊 Input Parameters

The model uses the following patient metrics:

* Temperature
* Oxygen Level
* Blood Pressure
* Comfort Score

---

## 🧠 Machine Learning Model

### Algorithm

* Decision Tree Classifier (Scikit-Learn)

### Prediction Classes

| Output | Meaning         |
| ------ | --------------- |
| 1      | Sent Home       |
| 0      | Keep Under Care |

### Workflow

1. User enters patient health metrics.
2. Frontend sends data to Flask backend.
3. Trained model processes the inputs.
4. Prediction is generated.
5. Result is displayed instantly.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Mehul-ui141/Patient-Classifier.git
cd Patient-Classifier
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
